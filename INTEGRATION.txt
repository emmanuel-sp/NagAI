Integration Steps:
1. Set NEXT_PUBLIC_API_URL env var (or update in lib/api.ts)
2. In each service file, find "// API call here" comments
3. Replace dummy data blocks (enclosed by "// Dummy data" comments) with apiRequest() calls
4. Update types in frontend/types/ if backend schema differs

Auth Flow:
- Token stored in localStorage as "authToken"
- Auto-injected in all requests via apiRequest()
- useAuth() hook handles route protection (requireAuth/redirectIfAuth options)

Example Integration:
Find this in any service file:
  // API call here
  // return await apiRequest<Goal[]>("/goals");

  // Dummy data
  await new Promise((resolve) => setTimeout(resolve, 300));
  return [...MOCK_GOALS];
  // Dummy data

Replace with:
  return await apiRequest<Goal[]>("/goals");

All service files use this pattern. All components import ONLY from services/ - no changes needed to components.
